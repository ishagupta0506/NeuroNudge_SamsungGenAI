# NeuroNudge

A simple Streamlit app.

## Quick Start

1. Install Python
2. Create a virtual environment
3. Install dependencies:
    ```
    pip install -r requirements.txt
    ```
4. Run the app:
    ```
    streamlit run app.py
    ```

**Optional:**  
Add your OpenAI API key: [Get API Key](https://platform.openai.com/api-keys)
